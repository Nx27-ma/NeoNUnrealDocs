# Test mode tool

The test mode tool allows you to quickly change a scene to a developer mode by switching a variety of toggles.

## Open the tool

To open the scene creator tool you have to click on the NeoN tab from there you can select the test mode toggles and start using it.  

![image](https://github.com/BAStudio/OperationStarfall/assets/91124861/789f4193-92cc-4b8e-a884-02aca843cc0f)

## How to add to the tool

### 1. open the path Assets/Editor/TestSwitchToggle and open layout.uxml<br /><br />
<img src="https://github.com/BAStudio/OperationStarfall/assets/91124861/e5044619-4b23-49ba-9915-061964821d24" width="400" height="150"><img/>

### 2. add a toggle in the layout by<br />
> 1. dragging the toggle from the controls in the Library in the bottom left corner into the hierarchy next to the other toggles.
> 2. give it a fitting name and describe what it does in the lable in the inspector at the right top corner.<br /><br />
<img src="https://github.com/BAStudio/OperationStarfall/assets/91124861/18df4403-a324-43a3-99ed-1af7b9ea31a4" width="800" height="400"><img/>

### 3. now add it in TestModeToggle.cs by
> 1. add a toggle variable<br /><br />

> 2. initialize the variable<br /><br />
